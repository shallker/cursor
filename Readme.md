
# cursor

  Get cursor infos in a text input

## Installation

  Install with [component(1)](http://component.io):

    $ component install shallker/cursor

## API

### cursor
#### .position
#### .height
#### .offsetTop
#### .offsetLeft
#### .pageTop
#### .pageLeft
#### .insert(String text)
#### .before(String text)
#### .after(String text)
